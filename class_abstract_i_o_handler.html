Detailed Description

Interface for communicating between logical and physically persistent data.

Input and output operations are channeled through a task queue that is managed by the concrete class implementing this handler. The queue of pending operations is only processed on demand. For certain scenarios it is therefore necessary to manually execute all operations by calling AbstractIOHanlder::flush().
